# RAVA8_RNG

The `RAV8_RNG` library implements the RAVA random number generator architecture for the ATmega32U4 
microcontroller.

Built on top of the `RAVA_RNG` library, it provides the hardware-specific functionality required 
for communication, device management, entropy acquisition, and random data generation, including:
- Message transport over USART and USB CDC communication interfaces using the `LUFA` Library
- Use of Timer0 and Timer1 for RNG acquisition by counting comparator pulses generated by the 
  comparator IC and avalanche noise circuitry
- Use of Timer3 to generate fixed-interval random-byte streams
- Use of Timer4 to generate the PWM boost switching signal 
- Device information services, including serial number retrieval, available RAM estimation, 
  internal temperature measurement, and supply voltage monitoring

In addition to the base functionality expected by the `RAVA_RNG` library, `RAVA8_RNG` also 
provides:
- EEPROM support for persistent storage of RNG configuration parameters
- Peripheral interfaces, enabling external hardware interaction through Timer3, ADC, External 
  Interrupt, Analog Comparator, Input Capture, and Pin Change Interrupt functionalities

The ATmega32U4 implementation is provided as the `RAVA8_RNG` library, serving as a foundation for 
users wishing to extend or customize the system functionality. To build a complete firmware 
application, a source file containing `main()` must be provided. Additionally, a header file named
`rava8_config.h` must be supplied to define the compile-time feature configuration.

For a reference implementation see 
[RAVA8 Firmware](https://github.com/gabrielguerrer/rng_rava_firmware).
